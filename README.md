**Plantilla utilizada:** Start Bootstrap

## Descripción breve

Este proyecto consiste en un portafolio web personal creado a partir de una plantilla de Bootstrap.  
El portafolio fue personalizado con información académica, habilidades, proyectos y datos de contacto.

La finalidad de esta actividad es practicar el uso de plantillas web, modificar su estructura y publicar el resultado en GitHub Pages.

## Plantilla utilizada

Para este proyecto se utilizó la plantilla:

**Start Bootstrap**

Link de descarga:

```text
https://startbootstrap.com/theme/resume
```

Esta plantilla fue elegida porque ya incluye una estructura tipo currículum/portafolio, con menú lateral, secciones organizadas y diseño responsivo.

## Framework CSS utilizado

Se utilizó **Bootstrap** como base de estilos.

No se utilizó Tailwind CSS ni frameworks de JavaScript como React o Vue.

La plantilla ya incluye Bootstrap dentro de su archivo principal de estilos:

```html
<link href="css/styles.css" rel="stylesheet" />
```


## Secciones del portafolio
- Inicio
- Sobre mí
- Habilidades
- Proyectos
- Contacto


## Proceso de creación

1. Se descargó la plantilla **Start Bootstrap - Resume** desde la página oficial.
2. Se descomprimió la plantilla y se abrió la carpeta del proyecto en Visual Studio Code.
3. Se conservaron los archivos principales de la plantilla:
   - `index.html`
   - `css/styles.css`
   - `js/scripts.js`
   - `assets/img/profile.jpg`
4. Se modificó el archivo `index.html` para cambiar el contenido original de la plantilla por información personal.
5. los menus se sobre escribieron 
6. Se realizaron pequeños ajustes visuales en el archivo `styles.css`.
7. Se agregó una interacción sencilla en `scripts.js` para mostrar un mensaje en la sección de contacto.
